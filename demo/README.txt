**Demo Page** 
This page showcases the individual components of the project. 

> ⚠️ **Note:** The full circuit demo is currently not available due to hardware issues with the battery causing brownout problems. The following demos focus on individual modules.

__Motion Detection Demo__
- Uses the PIR sensor connected to an LED.
- When motion is detected, the LED turns on.
- Demonstrates that the motion detection circuitry is working correctly.

__Camera Demo__
- Demonstrates the ESP32-CAM creating a web server.
- The camera feed can be accessed remotely via a laptop.
- Shows that the camera module and web server setup are functioning.
